# Kudos APP UI

## Description

A Simple app to give kudos to coworkers.

## Getting Started

### Dependencies
Install node packages if not already

```yarn install```

Or using npm

```npm i```


### Executing program
Fire up the server!

```npm start```

# Important Note

## For local dev make sure to install an Extension to bypass the CORS Block!
